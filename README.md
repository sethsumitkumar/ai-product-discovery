An AI-powered web application that helps users discover new product ideas instantly using OpenRouter API.

---

## 🔥 Features

* 🤖 AI-based product idea generation
* ⚡ Fast and real-time responses
* 🎨 Clean UI using Tailwind CSS
* 🌐 Built with Next.js 14

---

## 🛠️ Tech Stack

* Next.js
* React
* Tailwind CSS
* OpenRouter API

---

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/your-username/product-discovery.git
cd product-discovery
```

### 2. Install dependencies

```bash
npm install
```

### 3. Add environment variable

Create `.env.local` file and add:

```
OPENROUTER_API_KEY=your_api_key_here
```

### 4. Run the project

```bash
npm run dev
```
